# Filesystem Bundle

Composes filesystem-related tools and behaviors for Amplifier agents.

## Behaviors

- **apply-patch** — V4A diff-based file editing (the `apply_patch` tool)
- **filesystem** — Read, write, edit tools (future — not yet migrated)
- **full** — All filesystem tools together (future)

## Modules

- `tool-apply-patch` — V4A diff format file editing with two engine options

## Context

- `editing-guidance.md` — Teaches the model when to use which editing tool
